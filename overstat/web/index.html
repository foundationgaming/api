API Usage

Overstat is an unofficial Overwatch Stats API written in Go. Accessing its main API endpoint is extremely simple. Perform a GET request on the main lookup endpoint using one of the two following schemas.

For PC Stats

ovrstat.com/stats/pc/{region}/{username}

For Console Stats

ovrstat.com/stats/{platform}/{username}
